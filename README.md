# Игра 
   ## Общие положения:
   Игра представляет собой поле, в середине которого "генератор", который дает свет.
   Этот генератор надо питать - кидать в топку мусор, собранный с поля, и лут с монстров.
   Света в игре мало, он есть только под фонарными столбами, которые и питаются генератором(Почему так будет в описано в лоре). Столбов в игре ограниченное количество, изначально горят только некоторые, зажигать новые можно будет с помощью лута с монстров.
   Из темноты будут выходить монстры, их надо будет бить и получать лут.
   Лут делится на две категории:
   - питание генератора
   - ремонтные части для столбов
## Цель игры:
Открыть как можно больше столбов, открытие необходимо для выживания, так как мусор на
открытых участках будет кончатся.
## Конец игры:
Чиним столб(какой-нибудь конечный) и выходим к людям
## Лор
На солнечную систему напали фанатики, которые покланяются тьме. Они погасили солнце и пришли на землю убить все живое. Они не умеют продолжительное время существовать на свету.
ГГ повезло у него был генератор, с его помощью он питает столбы освещения и таким образом оберегает себя от большиснтва монстров. От большиснтва потому что некоторые монстры могут на неродолжительно время выбегать из темноты на свет под мантиями тьмы.
ГГ думает, что он остался один, но однажды он чинил столб освещения и из-за этого загарается еще несколько столбов, он идет по пути, освещенному столбами и натыкается на целый город.
ГГ не один на земле! хеппи энд.
